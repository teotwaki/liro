What is Liro?

Liro is a discord bot which follows in the footsteps of Lichess Bot without necessarily aiming to replace it. The main pain point that this bot is trying to solve is automating the role assignment for denizens of the IM Eric Rosen discord server. The project also aims to increase rating support to not only lichess accounts, but also chess.com accounts.

Liro has multiple features including..
  • Application Commands
  • Prefix Commands
  • Automatic Role Detection (heuiristics based)
  • Automatic role assignment based on ratings
  • OAuth account verification for lichess
  • Public dashboard
